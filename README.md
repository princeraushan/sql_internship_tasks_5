## ğŸ” Concepts Demonstrated

### Aggregate Functions
Used to perform calculations on multiple rows and return a single summarized value.

### GROUP BY
Groups rows that share common values and allows aggregation on each group.

### HAVING Clause
Used to filter aggregated results after grouping has occurred.

### WHERE vs HAVING
- `WHERE` filters individual rows before grouping
- `HAVING` filters groups after aggregation
